# Project Scope Statement

## Project Purpose
The purpose of this project is to become more familiar with HTML and create a personal introduction webpage. This project also helps demonstrate how planning ahead can make the development process more organized and efficient. This scope will guide the development of the HTML page, and the final webpage will strictly follow the elements defined in this document.

## Description
This project includes creating two files: a scope document and a single page HTML website. The website will introduce who I am, what I am studying, my interests, and my goals. The goal is to create a clear and structured page that is easy to read and meets all assignment requirements.

## Desired Results
The final webpage will include:

- A valid HTML document with a `<title>` element in the `<head>`
- A `<header>` section containing:
  - An `<h1>` with my name
  - An `<h2>` describing my major or professional interests
- A `<main>` section containing:
  - At least two `<p>` elements describing my goals and background
  - A `<ul>` with at least three `<li>` items listing my interests
- A `<footer>` section containing:
  - An `<a>` tag linking to my GitHub profile
- Proper use of semantic HTML tags including `<header>`, `<main>`, and `<footer>`

## Exclusions
This version of the project will not include CSS styling, JavaScript, multiple pages, images, or advanced design features. The focus is on HTML structure and proper semantic organization.
