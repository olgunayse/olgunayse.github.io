# Project Scope Statement

## Project Purpose
The main purpose of this project is to get more familiar with HTML and make a personal introduction page. This is to help understand how planning ahead can contribute to an easier and better organized building process.
## Description
I will create two documents including a scope document and a HTML website. This will provide an my introduction including what I am currently studying and interested in, my goals for the class, a list of some of my hobbies and a link to my GitHub profile. The focus will be to provide a structured format that makes it easy to read and understand as well as to make sure that each element is included (like headings, paragraphs, lists, links).
## Desired Results
At the end of this project, I will have created a clean and functional webpage providing all of the required components, including headings, paragraphs, an ordered or unordered list and a link to GitHub. The completed project will represent me and my understading of HTML and how to create a simple webpage with structure.
## Exclusions
This is a pretty basic project so it will not include advanced elements but instead on proper structured elements and working as intended.
